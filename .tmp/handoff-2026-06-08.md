## 第三步实施方案

### 步骤 1：迁移脚本 `execution/migrate_coaches.py`

```
from google.cloud import firestore
import requests

db = firestore.Client()
FIREBASE_API_KEY = "<from .env.local>"  # VITE_FIREBASE_apiKey

coaches = db.collection('coaches').where('password', '!=', None).get()
for doc in coaches:
    data = doc.to_dict()
    # 调用 Firebase Auth REST API signUp 创建账号
    resp = requests.post(
        f'https://identitytoolkit.googleapis.com/v1/accounts:signUp?key={FIREBASE_API_KEY}',
        json={"email": data['email'], "password": data['password'], "returnSecureToken": False}
    )
    if resp.status_code == 200:
        # 删除 password 字段，改 status 为 active
        doc.reference.update({'password': firestore.DELETE_FIELD, 'status': 'active'})
```

### 步骤 2：改 `admin.js` (~200 行处)

```
原来：addDoc(collection(db, 'coaches'), { email, displayName, password, status: 'pending', ... })
改为：
  1. fetch Firebase Auth REST API signUp 创建用户
  2. addDoc(collection(db, 'coaches'), { email, displayName, status: 'active', ... })  ← 无 password
```

Firebase Auth REST API: `https://identitytoolkit.googleapis.com/v1/accounts:signUp?key=<API_KEY>`
